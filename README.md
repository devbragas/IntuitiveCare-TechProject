# IntuitiveCare-TechProject
## Stack Principal utilizada
- BackEnd (Python com FastAPI):
Nesse teste, utilizei FastAPI por ser um framework orientado a APIs, podendo reduzir bastante o boilerplate, como também
para melhorar a legibilidade do código e facilitar a manutenção. A implantação de testes com documentação automática também foi outro motivo
que me levou a escolher essa stack, priorizando uma aplicação segura e rápida de ser feita.
